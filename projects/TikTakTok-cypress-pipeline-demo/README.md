# Tic-Tac-Toe Game

A React-based Tic-Tac-Toe game with 2-player and AI modes, built using Vite. Includes win/draw detection, responsive UI, and a simple random-move AI.

## Features

- 🎮 **2-player mode**: Play against a friend locally.
- 🤖 **AI mode**: Compete against a simple AI (random moves).
- 🏆 **Win/Draw detection**: Automatically detects game outcomes.
- 🔄 **Reset game**: Restart the game at any time.
- 🎨 **Responsive UI**: Clean and modern design with hover effects.
- ⏳ **AI delay**: Simulates "thinking" with a 500ms move delay.
